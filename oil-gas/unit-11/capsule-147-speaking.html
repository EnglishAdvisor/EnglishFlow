Oil Tank Containment Project
Your team is building an oil tank containment in the workshop. The containment will be delivered by truck to the site. You need to report progress to your manager using the Present Perfect.
Status — Tuesday
Task 1
✅
Build base
Task 2
✅
Assemble containment
Task 3
❌
Put tank in containment
Task 4
❌
Weld containment
Role-play — Wednesday Update
Student A telephones to ask about progress. Student B (in the workshop) answers.
Student A — Manager
Ask questions using Have they...?
• done the grinding?
• checked inside the tank?
• attached the cover?
• put the tank on the truck?
Student B — Workshop
Answer Yes, we have / No, we haven't.

Then tell A three things you have done today and three things you haven't done.
Useful Language
Have they built the base?
Yes, they have. / No, they haven't.
We've done the grinding.
We haven't attached the cover yet.
Has Simon checked the inside?
Yes, he has.
