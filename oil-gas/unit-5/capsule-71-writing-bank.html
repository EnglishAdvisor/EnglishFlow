Writing Bank 5 — Notes with warnings
Grammar focus: this / these
this — singular: this item / this report / this form
these — plural: these items / these reports / these forms
Please don't remove this from... — strong polite instruction
Please take this/these to... — polite request with direction
Warning: important safety information before instructions
Example note 1
Warning: Please don't remove this from the control room. This document contains sensitive position data. Please return this to the operations manager after reading.
Used for: keeping official documents in the right place
Example note 2
Please take these GPS readings to the survey team. Warning: Do not change any of these figures. These coordinates are final.
Used for: passing data between teams
1Fill in: this or these?
1. Please don't remove this from the survey vehicle. ✓ this (singular item)
2. Please take these reports to the manager. ✓ these (plural reports)
3. Warning: do not damage these geophones. ✓ these (plural)
4. Please store this equipment safely. ✓ this (singular)
5. These coordinates are confidential. ✓ these (plural)

Note-writing tips
Start with Warning: for safety info
Please + infinitive for polite instructions
this = one thing (singular)
these = more than one (plural)
Don't remove — do not take away
Take to — carry / deliver to someone
Return to — give back to
Store safely — keep in a safe place
Model note
Warning: Please don't remove this GPS receiver from the survey area.

This device is recording positions. Please return this to Faisal when the survey is complete.
