Present Continuous — The rule
am / is / are + verb-ing
Iam driving a tanker.
He / She / Itis loading the tanker.
We / You / Theyare working night shifts.
Heisn't driving now.
Is sheworking today?
Example from the unit
Marcos drives a fuel tanker for a distribution terminal in Campos. Right now he isn't driving — he's unloading petrol at a service station.
1Questions about the example
  • Does he drive a tanker? (Yes / No)
  • Is he driving a tanker now? (Yes / No)
  • What does he do? (his job)
  • What is he doing right now? (current action)
2Make sentences — use Present Continuous
  • The screen (show) an error message. → The screen is showing an error message.
  • The gauge (not work) properly.
  • The computer (make) a strange noise.
  • Some of the keys (not work).
  • The red light (flash).
3Telephone conversation — practise
A: Where are you now?
B: I'm in the tanker.
A: Are you driving?
B: No. I'm having lunch.

Make similar conversations using: you now / at the refinery / working / having a break · he now / outside / repairing something / looking for leaks · they now / at the plant / collecting data / testing pipes

Social English — greetings & progress

How's it going? — asking about work/progress
How are you getting on? — asking about learning/work

Grammar rules
When to use Present Continuous
✅ Actions happening right now
✅ Temporary situations
✅ Greetings about progress
Spelling: adding -ing
drive → driving
work → working
run → running
sit → sitting
Useful verbs
load / unload
check / test
repair / fix
flash / show
collect / monitor
